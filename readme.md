SM2RAIN\_ASCAT\_reader
================

GitHub Documents
----------------

Simple code for reading the SM2RAIN-ASCAT .tif dataset (<https://zenodo.org/record/2615279>) in R.

Required packages - CRAN
------------------------

-   dplyr
-   ggplot2
-   gridExtra
-   lubridate
-   raster
-   rgdal
-   scales
-   tidyr
